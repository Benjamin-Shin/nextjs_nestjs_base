## Development Stack

* Language : Typescript
* Database : SQLite
* Backend : Nest.js
  * ORM: Prisma
  * Tools: Swagger
* Frontend : Next.js
  * UI : Chakra UI, TailwindCSS
  * Tools: i18next


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
