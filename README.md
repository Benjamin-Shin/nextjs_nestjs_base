# Project based on Next.js + Nest.js 

* Blog Link : https://endev.tistory.com/153

## Development Stack

* Language : Typescript
* Database : SQLite
* Backend : Nest.js
  * ORM: Prisma
  * Tools: Swagger
* Frontend : Next.js
  * UI : Chakra UI


## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
